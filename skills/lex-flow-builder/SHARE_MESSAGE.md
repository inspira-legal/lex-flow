Tenho o prazer de anunciar que a skill **lex-flow-builder** está disponível para uso no Claude Code!

### O que é?
Uma assistente de IA que te guia na criação completa de workflows LexFlow, desde a coleta de requisitos até as instruções de deploy.

### Como instalar?

1. Abra o terminal e execute:
```bash
git clone https://github.com/inspira-legal/lex-flow.git
cp -r lex-flow/skills/lex-flow-builder ~/.claude/skills/
```

2. Reinicie o Claude Code

3. Pronto! Só começar a usar.

### Como usar?

Abra o Claude Code e diga algo como:
- "Quero criar um workflow LexFlow para processar PDFs"
- "Preciso automatizar envio de emails com dados de CSV"
- "Crie um fluxo para validar documentos"

A skill automaticamente:
- Faz perguntas para entender seus requisitos
- Cria um design visual do fluxo
- Implementa o código YAML
- Valida a estrutura
- Gera documentação explicativa
- Fornece instruções de deploy

## Versão para Wiki/Confluence

### LexFlow Builder - Skill para Claude Code

**Status:** ✅ Disponível
**Versão:** 1.0.0
**Última atualização:** 2026-03-09

#### Descrição
Skill de IA que assiste na criação completa de workflows LexFlow através do Claude Code, cobrindo desde discovery de requisitos até instruções de deploy.

#### Pré-requisitos
- Claude Code instalado
- Git
- Acesso ao repositório: https://github.com/inspira-legal/lex-flow

#### Instalação

1. **Clonar e instalar a skill:**
```bash
git clone https://github.com/inspira-legal/lex-flow.git
cp -r lex-flow/skills/lex-flow-builder ~/.claude/skills/
```

2. **Verificar instalação:**
```bash
ls ~/.claude/skills/lex-flow-builder/SKILL.md
```

3. **Reiniciar Claude Code**

#### Como usar

##### Ativação Automática
Mencione palavras-chave no chat:
- workflow
- lex-flow
- fluxo
- automação
- n8n

##### Ativação Manual
```
/skill lex-flow-builder
```

##### Exemplo de uso
```
User: "Preciso criar um workflow que processa CSVs e envia notificações"

Claude (usando skill):
1. Faz perguntas sobre inputs/outputs
2. Cria design visual do fluxo
3. Implementa YAML completo
4. Valida estrutura e opcodes
5. Gera documentação
6. Fornece instruções de deploy (CLI/programático/web)
```

#### Funcionalidades
- ✅ Discovery guiado de requisitos
- ✅ Design visual de fluxos
- ✅ Implementação YAML automática
- ✅ Validação de estrutura
- ✅ Geração de documentação
- ✅ Instruções de deploy

#### Recursos Disponíveis
- **reference.md** - Catálogo de opcodes LexFlow

- **README.md** - Documentação completa

#### Troubleshooting

| Problema | Solução |
|----------|---------|
| Skill não encontrada | Verificar caminho `~/.claude/skills/lex-flow-builder/` |
